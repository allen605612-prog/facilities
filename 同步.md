# Claude Code 跨電腦同步指南

## GitHub Repo
https://github.com/allen605612-prog/claude-config

---

## 在新電腦還原設定

### 步驟一：安裝 Claude Code
```bash
# Windows PowerShell
irm https://claude.ai/install.ps1 | iex
```

### 步驟二：下載設定
```bash
git clone https://<你的token>@github.com/allen605612-prog/claude-config.git
```

### 步驟三：複製到正確位置
```bash
cp claude-config/CLAUDE.md ~/.claude/
cp claude-config/settings.json ~/.claude/
mkdir -p ~/.claude/commands
cp claude-config/commands/* ~/.claude/commands/
```

重新開啟 Claude Code，設定即還原完成。

---

## 這台電腦更新設定後，同步到 GitHub

每次新增或修改設定後執行：

```bash
cd /c/Users/user/claude-config
cp ~/.claude/CLAUDE.md .
cp ~/.claude/settings.json .
cp ~/.claude/commands/* commands/
git add .
git commit -m "更新設定"
git push
```

---

## 另一台電腦拉取最新設定

```bash
cd claude-config
git pull
cp CLAUDE.md ~/.claude/
cp settings.json ~/.claude/
cp commands/* ~/.claude/commands/
```

## 首次設定 GitHub（只需做一次）

### 步驟一：設定 Git 身份
```bash
git config --global user.name "allen605612-prog"
git config --global user.email "allen605612@gmail.com"
```

### 步驟二：建立本地 repo 並複製設定檔
```bash
mkdir -p /c/Users/user/claude-config/commands
cp ~/.claude/CLAUDE.md /c/Users/user/claude-config/
cp ~/.claude/settings.json /c/Users/user/claude-config/
cp ~/.claude/commands/* /c/Users/user/claude-config/commands/
```

### 步驟三：初始化 Git 並上傳到 GitHub
```bash
cd /c/Users/user/claude-config
git init
git add .
git commit -m "Claude Code 個人設定"
git branch -M main
git remote add origin https://<你的token>@github.com/allen605612-prog/claude-config.git
git push -u origin main
```

### 取得 GitHub Token
1. 登入 GitHub → 右上角頭像 → **Settings**
2. 左側最下方 → **Developer settings**
3. **Personal access tokens** → **Tokens (classic)**
4. **Generate new token (classic)**
5. 勾選 **`repo`** 權限，Expiration 選 **No expiration**
6. 點 **Generate token**，立刻複製（離開頁面後看不到）

---

## 注意事項
- Token 請妥善保管，不要公開分享
- Token 可在 GitHub → Settings → Developer settings → Personal access tokens 重新產生
- plugins 需在新電腦手動重新安裝
